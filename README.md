# Clipper ✂️ (currently building...)

A minimal Dynamic Island–style floating UI for macOS, built using AppKit + SwiftUI.

https://github.com/user-attachments/assets/e31aad22-f018-480c-b497-7372fa887317

### ✅ Whats done:
- Floating top-center island
- Always visible (on top of all apps)
- Smooth expand / collapse
- Hover to preview
- Click to pin open
- Click outside / Esc to close
- Minimal pill (closed) + expanded view

### 🚀 Coming Next
- Timer (live activity)
- Media controls
- Smart notifications
- COPY & PASTE HISTORY BOARD (most important)
  

### 🛠️ How to run
- Clone the repo
- Open in Xcode
- Build & run
